# X-Novel
一款基于Taro的小说小程序，使用了最新的Hooks方式和基于此方式的Redux，小说API接口提取自追书神器APP，仅供学习交流使用，如有侵权，请及时联系。

### 用到的技术
+ Taro
+ Hooks
+ Redux
+ reselect
+ Sass

### 运行
```
+ git clone https://github.com/Jerry-1123/X-Novel.git
+ cd X-Novel
+ npm install
+ npm run dev:weapp
+ 使用测试AppId，勾选"不校验和法域名"
```
